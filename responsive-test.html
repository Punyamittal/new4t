6. Breakpoint Testing Instructions

Test the following breakpoints in Chrome DevTools:

  • 320px - Extra small phones (portrait)
  • 375px - iPhone SE, iPhone 12 mini
  • 414px - iPhone 12 Pro Max
  • 768px - Tablets (portrait)
  • 1024px - Tablets (landscape), small laptops
  • 1440px - Desktop screens

Check for:

  • No horizontal scrolling
  • Proper text scaling
  • Touch-friendly button sizes
  • Responsive grid layouts
  • Images scaling correctly
  • Navigation adapting properly
